use strict:

The used of 'use strict'. The 'use strict' always will be in first line of your code also it will help you to find bugs and errors. When you used "classes and "modules" you don't have to use it.

Variables:

Variables are used to store data or as a "name storage". We can declare multiple variables in one line When we change the value of a variable the old one is remove from the variables. we can declare two variables and copy data from one into the another. There are a lot option on how you want to declare variables of is depends of preferences.

Var:

Var is a old school variables that developers don't use anymore.

Let:

Is used now instead of var. Let lets you change data. For best practice is better to used const instead of let because of bugs or mutation.

Constant:

Const is a variable that it wont change data and because of that you will have less mutation and bugs.

Data Types:

In JS a value is a certain type and there are 8 basic types of data.

Number:

The number type represents both integer and floating numbers.

Example: let n = 13;

NaN

Represent a computational error.

Example: let x = "lol" / 2; // NaN

String

Store a string of characters

Example:

We have 3 types of quotes.

let x = "Hello"

let x = 'Hello'

let x = `Hello`

Boolean(logical type)

The boolean type only have true or false.

Example:

let x = true;

let x = false;

Null

Is a special value that does not belong to any type describe. Also represent "nothing" or "empty".

Example

Let x = null;

Undefined

Is similar to null but the meaning of undefined is that a variable is declare but not assigned

Example:

let x; // shows Undefined

Objects

The object is different. All the other types are called "primitive" because they can only have one thing. Objects can store collections of data and more complex entities.
